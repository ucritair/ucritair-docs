Logs


Air quality data is logged at a user-configurable rate. A higher logging rate will fill the Î¼CritAir's storage space more quickly and also consume more battery. Logs are intended to persist indefinitely until cleared. If logs are cleared, dashboard features that rely on logs will be unavailable until a handful of new logs are made.

Logged Metrics

The logged metrics are:

  • CO2
  • PM 2.5
  • PN 10.0
  • NOx Index
  • VOC Index
  • Temperature
  • Relative Humidity
  • Pressure

Logging Settings

Settings related to logging can be found under MENU / SETTINGS / SYSTEM. The SET CLOCK + LOG RATE page allows you to set the period of time in between new log entries. Under this page one can also find the setting for logging NOX/VOC every N entries. By default, NOX/VOC logging is disabled because it consumes extra battery life.

The ERASE ON DEVICE LOGS option will delete all of the logs currently stored on your device. Be careful with this option!
